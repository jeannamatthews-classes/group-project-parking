Demo – Parking App JSON (Testing Guide)

This folder contains the demo version of the Parking App's Data Transfer Process. It is used for internal data transfer testing by group members only.


How to run:
- 
1) Open server.py in VS Code.
2) Keep it running during testing.
3) Open project in android studio and wait for gradle to sync (took 10+ minutes the first time for me).
4) Click run.
5) Will work locally (Current refresh is about 5 seconds otherwise force refresh with button input).
